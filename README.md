**Application Health Checker**
This Python script monitors the uptime of an application by checking HTTP status codes. It determines whether the application is functioning correctly ("UP") or experiencing issues ("DOWN")
**Features**
Checks the HTTP status code of a specified URL.
Prints the current status of the application ("UP" or "DOWN (status_code)").
**Requirements**
  Python 3.x
  requests library
**Clone the repository:**
git clone https://github.com/rohitutekar123/Application-Health-Checker.git
**Install requriments**
  pip install requests
**Run the python file **
python application_health_checker.py
